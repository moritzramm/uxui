Define

Put together the information you have created and gathered during the Empathise stage. Analyse your observations and synthesise them in order to define the core problems that you have identified up to this point. Define the problem as a problem statement in a human-centred manner.

Methodology and Tools

  • Affinity Diagram
  • Empathy Map
  • User Persona(s)
  • Story Board
  • User Journey
  • JdBT - Jobs to be done
  • Problem Statement
  • 5 'W's - Who, What, Where, When and Why
  • HowMightWe Statement
  • Pain Pointe
